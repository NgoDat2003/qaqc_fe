# Maycha QA/QC Platform Topology

## Pages
1. **Login Page** (`/login`): Current authentication gateway.
2. **Dashboard** (`/dashboard`): Overview of QA/QC operations.
3. **Criteria Management** (`/master-data/criteria`): Setup of scoring criteria.
4. **Checklist Forms** (`/master-data/checklists`): Form designer/editor.
5. **Brands & Stores** (`/master-data/locations`): Management of organizational structure.
6. **Users & roles** (`/settings/users`): RBAC management.
7. **Audit Execution** (`/operations/audit`): The core wizard for auditing.
8. **Action Plans** (`/operations/action-plans`): Tracking violations and remediation.

## Global Shell
- **Sticky Sidebar**: Left-side navigation with collapsible groups.
- **Top Header**: Breadcrumbs, Notifications, User Profile.
- **Theme**: Ant Design based, Primary Gold color.

## Tech Stack (Clone)
- **Next.js 16 + React 19**
- **Tailwind CSS v4**
- **Shadcn UI** (Replacing Ant Design for better aesthetics)
- **Lucide Icons**
